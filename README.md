# react-native-mqtt-light

Lightweight MQTT client for react-native

## Installation

```sh
npm install react-native-mqtt-light
```

## Usage

```js
import MqttLight from "react-native-mqtt-light";

// ...

const result = await MqttLight.multiply(3, 7);
```
